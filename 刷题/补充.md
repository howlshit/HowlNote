## 1

二分查找：不存在返回-1，存在多个返回第一次出现的下标（可向前遍历或继续二分）

```java
public class BinarySearch {
    public int getPos(int[] A, int n, int val) {

        int left = 0;
        int right = n - 1;
        int mid = 0;
        
        while(left <= right){
            
            mid = (left+right) >> 1;
            
            if(A[mid] > val){
                right = mid - 1;
            }else if(A[mid] < val){
                left = mid + 1;
            }else{            // 这里相等
                right = mid - 1;  // 向前二分的重点
            }
            
        }
        return A[left] == val ? left : -1;
    }
}
```
